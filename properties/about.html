Semantic Integration Workbench v3.2.1.2
New Features include:
  • GF 10036 - Support for models created in Argo UML. Please note that users may experience problems
    re-importing files in ArgoUML if the model contains one or more empty diagrams
  • GF 8889 - SIW can now generate default GME Tags and save them in the input XMI as tagged values
  • GF 8890 - SIW now allows reviewing GME tags with the option not to display GME tags
  • GF 8893 - New SIW validators to verify format of GME namespaces. SIW also throws a warning for missing GME tags
  • GF 8894 - SIW has a new GME Cleanup feature which removes all GME tagged values from an input file
